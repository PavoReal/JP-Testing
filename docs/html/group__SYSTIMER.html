Detailed Description

System Timer is a memory-mapped peripheral. It features a 64-bit free-running counter that runs at 1 MHz and four separate “output compare registers” that can be used to schedule interrupts. However, two output compare registers are already used by the VideoCore GPU, leaving only two available for the ARM CPU to use.
